## 실행 방법

Python Version: 3.12.7

### 패키지 설치
```
pip install requirements.txt
```

### 환경변수 설정
```
export SECRET_KEY=<YOUR SECRET KEY TO JWT ENCODE>
export DB_USER=<DB USER>
export DB_PASSWORD=<DB PASSWORD>
```

### 실행
```
python app.py
```
